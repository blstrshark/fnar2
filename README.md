How To Open

1: Unzip the file.
2: If Apple doesn't open the game because it cannot scan for malware:
	- go to system settings
	- go to privacy and security
	- scroll down and click "run anyway" for the game file
3: When the game opens properly, the Apple terminal will be opened. You'll need to wait for up to a minute once this happens.
4: If the wait is over, the game will launch. Have fun!

Controls

Q/W - Lights
Z/X - Doors
B - Battery
Click on Rael (the baby) - Make him 10% happier (necessary to avoid a grouchy baby.)
1-9 - Cameras
